# P&L Dashboard — RM GROUP

Управленческий отчёт о прибылях и убытках (P&L) за январь–сентябрь 2025.

Возможности:
- 6 вкладок: Обзор · Денежные потоки · Поставщики · Продажи · Детально · Выводы
- KPI-карточки со спарклайнами и % изменения
- Переключатель тем (светлая/тёмная)
- Экспорт в PDF / печать

Технологии: HTML, CSS, Chart.js 4.4.1 (CDN). Данные встроены в файл — без бэкенда.

Запуск: откройте `index.html` в браузере или через GitHub Pages.
